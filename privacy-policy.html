Privacy Policy

Binance P2P LKR Rate Live Update — Chrome Extension

Effective Date: July 2026

Summary: Infiax (Pvt) Ltd values your privacy. This extension does NOT collect, track, transmit, or store any personal data, passwords, credentials, or browsing history. All operations occur locally in your browser.

1. Information Collection and Use

We believe in strict data privacy. Binance P2P LKR Rate Live Update does not collect any personally identifiable information (PII) from its users.

  • No Personal Identifiers: We do not collect names, email addresses, IP addresses, or location data.
  • No Financial Credentials: The extension does not ask for or access your Binance login credentials, private keys, or wallet passwords.
  • No Browsing History: The extension does not monitor, track, or record your web browsing activity.

2. Local Data Storage

The extension uses Chrome's native chrome.storage.local API strictly for the following functional purposes:

  • User Preferences: Storing your preferred target order limit amount (e.g. Rs. 50,000) so you don't have to re-enter it every time.
  • Local Caching: Caching the latest exchange rates temporarily for fast offline rendering.

This data stays 100% on your personal device and is never uploaded to any external server.

3. Third-Party API Requests

To display live exchange rates, the extension sends direct POST network requests to Binance's public JSON API endpoint (https://www.binance.com/bapi/c2c/v2/friendly/c2c/adv/search).

  • These API requests contain only public currency query parameters (e.g. fiat: "LKR", asset: "USDT").
  • No user identification headers or personal data are included in these API calls.

4. Policy Updates

We may update this Privacy Policy from time to time to reflect changes in our practices or Chrome Web Store policies. Any updates will be posted on this page with an updated effective date.
